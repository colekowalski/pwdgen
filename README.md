# pwdgen

A simple and secure random password generator

## Example
```shell
$ pwdgen --count=4 --length=32
XE!Js42HvD8gvU04^SCb&70lNqolUt0L
xK&0sBfA09aS!jT3yTE*dXZlR8QBSL6G
6rxIXV@paiAMRW#Y#cgHfQ6mJLi!cpOl
UN*&VMo^wte76J*WAd06o*YAaO%KbqNq
```
